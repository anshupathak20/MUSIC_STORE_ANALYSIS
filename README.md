#  Music Store Analysis using PostgreSQL

## Project Overview

This project analyzes a Music Store database using PostgreSQL. The objective is to solve real-world business problems by writing SQL queries and extracting meaningful insights from the data.

## Objectives

- Analyze customer purchasing behavior.
- Identify top-selling artists and genres.
- Find high-value customers.
- Generate business insights using SQL.

## Tools & Technologies

- PostgreSQL
- SQL
- Git
- GitHub

##  Business Questions Solved

- Who is the senior most employee?
- Which countries generate the highest revenue?
- Which city has the highest invoice total?
- Who is the best customer?
- Top Rock music listeners.
- Top Rock artists.
- Tracks longer than average duration.
- Customer spending by artist.
- Most popular music genre by country.
- Highest spending customer by country.

  
# #How to Run

1. Import `MUSIC_STORE_ANALYSIS_PROJECT.sql` into PostgreSQL.
2. Open `MUSIC_STORE_ANALYSIS_PROJECT_QUERIES.sql`.
3. Execute the queries one by one.
4. Analyze the results.
